User Journey
Tourist (นักท่องเที่ยว)
1. เข้าเว็บไซต์ LHKEM
2. สมัครสมาชิกหรือเข้าสู่ระบบ
   1. เลือก Role: Tourist ตั้งแต่หน้าสมัคร
3. เข้าหน้าแรก ดูรายชื่อชุมชนทั้งหมดในแพลตฟอร์ม
4. เลือกคลิกชุมชนที่สนใจ
5. อ่านข้อมูลชุมชน (ประวัติ จุดเด่น รูปภาพ)
6. ดูรายการ Workshop / Event ของชุมชนนั้น
7. คลิก Workshop ที่สนใจ
   1. อ่านรายละเอียด Workshop
   2. วันเวลา
   3. จำนวนที่นั่ง
   4. ร้านผู้จัด
   5. กดปุ่ม “ลงทะเบียน”
   6. ระบบแสดงช่องทางติดต่อร้านค้า (เช่น LINE)
   7. แอดไลน์ร้านค้าเพื่อไปชำระเงินหรือยืนยันการเข้าร่วมกิจกรรม
8. ดูแผนที่ชุมชน
9. ดูตำแหน่งร้านค้า
10. ดูเวลาเปิด–ปิด และรายละเอียดร้านที่สนใจ
11. รับอีเมลแจ้งเตือน หากมีข่าวสารหรือการอัปเดตจากร้านค้า/กิจกรรม


Platform Admin (แอดมินแพลตฟอร์ม)
1. สมัครสมาชิกหรือเข้าสู่ระบบ
   1. เลือก Role: Platform Admin ตั้งแต่หน้าสมัคร
2. เข้าหน้า Platform Admin Dashboard
3. ดูภาพรวมแพลตฟอร์ม
   1. จำนวนชุมชน
   2. จำนวนผู้ใช้งาน
4. เลือกเมนูจัดการชุมชน
5. สร้างชุมชนใหม่ในแพลตฟอร์ม
   1. ระบุชื่อชุมชน (ขั้นต่ำ)
   2. ยังไม่ต้องกรอกรายละเอียดเชิงเนื้อหาให้ชุมชนั้นๆกรอกเอง
6. บันทึกและเปิดใช้งานชุมชนในระบบ
7. ตรวจสอบและดูแลสิทธิ์การเข้าถึงของผู้ใช้งานแต่ละระดับ
8. ปรับสถานะ Commu ตามความเหมาะสม
   1. เปิด / ปิด
   2. แสดง / ซ่อน
9. ทำหน้าที่ดูแลโครงสร้างแพลตฟอร์มโดยรวม
   1. ไม่จัดการ workshop หรือข้อมูลภายในชุมชน


________________


Community Admin (แอดมินชุมชน)
1. สมัครสมาชิกหรือเข้าสู่ระบบ
   1. เลือก Role: Community Admin ตั้งแต่หน้าสมัคร
2. เข้าหน้า Community Admin Panel
3. จัดการข้อมูลชุมชนของตนเอง
   1. ประวัติชุมชน
   2. รูปภาพ
   3. ช่องทางติดต่อ
4. ดูรายชื่อร้านค้าในชุมชน
5. ดูรายการ Workshop / Event ที่ร้านค้าสร้าง
6. ตรวจสอบรายละเอียด Workshop
7. อนุมัติ Workshop หรือส่งกลับให้ร้านค้าแก้ไข
8. สร้างหรือจัดการ Event ระดับชุมชน (ถ้ามี)
9. ปรับสถานะ Event หรือ Workshop ตามความเหมาะสม
   1. เปิด / ปิด
   2. แสดง / ซ่อน
   3. 

Shop (ร้านค้า)
1. สมัครสมาชิกหรือเข้าสู่ระบบ
   1. เลือก Role: Shop ตั้งแต่หน้าสมัคร
2. เข้าหน้า Shop Dashboard
3. สร้างหรือแก้ไข Shop Profile
   1. ชื่อร้าน
   2. รูปภาพ
   3. ข้อมูลติดต่อ (LINE / Facebook / โทร)
   4. เวลาทำการ
4. สร้าง Workshop ใหม่
5. กรอกรายละเอียด Workshop
   1. ชื่อกิจกรรม
   2. วันเวลา
   3. จำนวนที่นั่ง
6. บันทึก Workshop และส่งรอการอนุมัติจาก Community Admin
7. ตรวจสอบสถานะการอนุมัติ Workshop
8. เมื่อ Workshop ถูกอนุมัติ
   1. ดูรายชื่อผู้ลงทะเบียน
   2. ตรวจสอบจำนวนที่นั่งคงเหลือ
9. ปิดรับสมัคร Workshop เมื่อครบจำนวนที่กำหนด


สรุป
* Tourist: สมัครเลือก Tourist > ดูชุมชน > เลือก Workshop > ลงทะเบียน > ติดต่อร้าน > ดูแผนที่
* Platform Admin: สมัครเลือก Platform Admin > สร้างชุมชน > ตั้ง Community Admin > ดูแลสิทธิ์
* Community Admin: สมัครเลือก Community Admin > จัดการข้อมูลชุมชน > อนุมัติ Workshop / Event
* Shop: สมัครเลือก Shop > จัดการร้าน > สร้าง Workshop > ดูผู้สมัคร
